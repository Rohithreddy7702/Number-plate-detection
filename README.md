# 🚗 Number Plate Detection System

An AI-powered web application that automatically detects and reads vehicle number plates from images using Computer Vision and Deep Learning.

## 🔧 Tech Stack
- **Python** — Core programming language
- **OpenCV** — Image processing & plate detection
- **EasyOCR** — AI-based text recognition (Deep Learning)
- **Flask** — Web framework & dashboard
- **SQLite** — Database for storing detections
- **Render** — Cloud deployment

## ✨ Features
- Upload any car image and detect the number plate instantly
- AI reads plate text with confidence score
- Full detection history with search
- Blacklist alert system — flags stolen/suspicious vehicles
- Visual statistics with charts
- Live deployment on cloud
